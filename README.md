# React Native with Redux and React Navigation

#### Getting Started
1) download or clone repository
2) npm install
3) 'react-native run-ios' for iOS or 'react-native run-android' for Android

#### Boilerplate structure
./src
+-- actions
|   +-- index.js
|   +-- types.js
+-- components
|   +-- Main.js
+-- navigation
|   +-- Navigator.js
+-- reducers
|   +-- index.js
|   +-- sample.js
+-- store
|   +-- index.js
